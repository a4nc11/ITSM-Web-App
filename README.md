## ðŸ›  Project Overview
A **web-based IT Service Management (ITSM) system** designed for organizations to manage IT issues efficiently.  
- Employees can log hardware, software, and network issues.  
- IT support staff can manage, prioritize, and resolve tickets.  
- Role-based access ensures secure and organized operations.  

This project showcases **full-stack development, database management, testing, and basic cybersecurity practices**, making it a strong flagship project for IT graduate portfolios.

### **User Side**
